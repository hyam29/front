함수연습

                    0987654321
    0         1         2         3
    01234567890123456789012345678901
    ABCDEFGHIJKLNMOPQRSTUVWXYZABCopq

length : 객체(문자열)의 길이

slice(a,b) : JS. 지정된 문자열(a부터 b 바로 앞 까지)을 추출해준다. (subString과 비슷)

substr() : subString을 대신해서 -인덱스 출력

replace(a, b) : a를 b로 치환 (대소문자 구분)

replaceAll(a, b) : 모든 a를 모두 b로 치환 (대소문자 구분)

toUpperCase() : 대문자 치환

toLowerCase() : 소문자 치환

concat(a, b) : a와 b를 결합

trim() : 양쪽 공백을 삭제한다.

trimStart : 문장 앞쪽 공백 삭제

trimEnd : 문장 앞쪽 공백 삭제

모든 공백 삭제는 replaceAll(" ","") 사용

charAt(); : 값 한개 뽑아 출력

charCodeAt() : 입력한 문자의 ASCII 코드 출력

split() : 구분자를 기준으로 잘라준다. 잘라진 각각의 요소는 배열로 저장된다.

indexOf() : 특정문자의 위치값을 찾아준다. (인덱스번호 출력)

lastIndexOf() : 특정문자의 위치를 뒤에서부터 찾아서 출력 (인덱스번호 출력)

search() : JS. 특정문자의 위치값을 특정문자의 첫번째 글자 인덱스로 출력 (찾는값이 없다면 -1 출력)

match() : 찾는 문자열이 있으면 '문자열'을 반환하고, 없으면 '공백'을 반환

includes() : 찾는 문자열을 포함하고 있으면 true, 없으면 false를 반환
