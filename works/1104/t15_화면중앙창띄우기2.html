화면중앙에 새창띄우기(부모창)

텍스트박스에 윈도우 가로크기와 세로크기를 입력받아서
버튼을 클릭하면 화면 중앙에 사용자가 지정한 크기의 새창(t15_child.html)을 띄우시오.
단, 최소 창의 크기는 (300px * 200px)

윈도우 가로크기 : 윈도우 세로크기 : 

        : 새창에 사용되는 이름(name)?
          _blank : 새창(기본값), _parent : 부모프레임에 열림, _top : 로드된 프레임셋을 다시 대체함, _self : 자신창에 띄움
          (_top 많이 사용하니 꼭 알아두기! top은 화면 전체를 뭉쳐서 다시 띄움)

        : 새창에 사용되는 옵션(option)?
          width = : 창의 폭
          height = : 창의 높이
          left = : x축 위치
          top = : y축 위치

          auto가 디폴트!
          fullscreen=(yes/no/auto) : 전체화면모드 (현재지원X)
          titlebar=(yes/no/auto) : 제목표시줄
          location=(yes/no/auto) : 주소표시줄의 출력여부를 결정
          menubar=(yes/no/auto) : 메뉴표시줄의 표시여부를 결정
          resizable=(yes/no/auto) : 창의 크기 제어
          scrollbars=(yes/no/auto) : 스크롤바 표시여부를 결정
          toolbar=(yes/no/auto) : 툴바(도구상자)의 표시여부 결정
